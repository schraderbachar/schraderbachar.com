How I got here

Discovering My Passion

I found my passion when I created a lifting website to help people with their form on certain movements. To give access to information to people who didn't necessarily have money to pay for a coach. It was horrible at first, as I was completely self-taught in 2018.

I graduated high school in 2019 and spent a year in Chile as an exchange student before coming to college. Then Covid hit the spring of 2020. I still had that passion to make my views come to life and solve problems for everyone's benefit, so instead of wasting away in my room, I decided to learn more about web development.

Embarking on a Learning Journey

I started with youtube tutorials building to-do apps, lists, and different javascript frameworks. I also took the Cs50 course from Harvard that was free on youtube. I really love coding and wanted to improve.

Creating Practical Solutions

Then I started making web apps to solve real problems. The first one I did completely on my own was a workout generator. My host sisters and I wanted to work out, but they didn't like the idea of doing the same routine every week, so I made a workout generator using Vue.js to have it spit out 5 exercises for us to do; that way, it was fair to all of us, and we couldn't complain about which exercises were and weren't being picked. They and I really liked it, and they were amazed by it, and it felt good to be able to make something that solved a real-world issue, at least in my world..

Fast forward to College, I got a job creating a memorial site and working under financial aid as a web developer; lots of real-world problems to solve there, and I love every day I go in; it doesn't feel like work.

Pushing Boundaries with Web Apps

Fast forward to recently, I built a web app that uses the Spotify API to generate a certain subgenre of an EDM song because when I was listening to music at festivals, I always wondered about the specific sub genre; I couldn't find one, so I made one. I don't see myself working in anything else except web dev related, whether it be front-end, backend, or full-stack. I love it all

ISU Course Scheduler

TypeScript | Python | React | TailwindCSS

The Iowa State Ivy College of Business wants a system that predicts student enrollment in future course sections (the upcoming semester) to determine what courses to offer and at what time. They also want to show which professor teaches a certain class at a certain time to schedule the professors as well. As a team of 5, we built a course scheudling app that lets the IVY admins, upload past student and class data (13 MB), and using ML algorithms, it returns the probablity that a certain class at a certain time will be taken. This allows for more efficient scheduling, and better results for teachers and students

EDM Genre Generator

React | Express | Node.js | Spotify API

My EDM Genre Generator is an innovative web application built using React, Express, Node and Spotify's API. I came up with the idea while a friend was playing at a rave. By simply inputting a Spotify song link, the application extracts data to accurately predict the genre and subgenre of the song. Through advanced algorithms and analysis of various song characteristics, including tempo, instrumentation, and energy level, precise genre and subgenre predictions are made.The implementation of cookie generation and authentication ensures a secure and personalized user experience.
